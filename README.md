# Nel's Closet

## Description

E-commerce shopping app with login and logout capabilities. Uses api to display products for browsing.

## Tech Stack

    - MongoDb
    - React
    - JavaScript
    - CSS

## Getting Started

### Executing program

- After cloning run:

```
npm install
npm start
```
